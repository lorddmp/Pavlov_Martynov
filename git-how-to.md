# Welcome to Markdown

Markdown is a lightweight markup language used for formatting plain text.

## Basic Formatting

You can make text bold or *italic*.

### Lists

Here is an unordered list:
*   Item one
*   Item two

Here is an ordered list:
1.  First step
2.  Second step

### Links and Code
